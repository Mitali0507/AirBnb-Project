# Airbnb Market Analysis Dashboard

## Overview
An interactive Tableau dashboard analyzing Airbnb listing data to uncover pricing patterns, competitive landscape, and revenue trends across different zip codes and property types.

## Dataset
- **Source:** Kaggle - Airbnb Dataset
- **Scope:** Airbnb listings with pricing, location, and property details
- **Tools Used:** Tableau Public, Excel

## Key Visualizations

### 1. Average Price per Bedroom
- Bar chart showing pricing trends from 1-bedroom to 6-bedroom properties
- Reveals price scaling: Studio ($95.4) to 5-bedroom ($513.5)
- Identifies optimal pricing strategy by property size

### 2. Price Analysis by Zipcode
- **Geographic Heatmap:** Visual representation of pricing distribution across zip codes
- **Bar Chart:** Comparative analysis of average prices across 15+ zip codes
- Highlights premium locations (98109, 98121) vs. budget-friendly areas

### 3. Market Competition Analysis
- Distinct count of bedroom listings showing market saturation
- 1-bedroom (886 listings) and 2-bedroom (258 listings) dominate the market
- Identifies underserved segments (5-6 bedroom properties)

### 4. Revenue Trend - 2016
- Line chart tracking revenue performance throughout the year
- Shows seasonal patterns and peak booking periods
- Revenue peaked around May-July period

## Insights
- 1-2 bedroom properties have highest competition
- Zip codes 98109 and 98117 command premium pricing
- Clear correlation between bedroom count and average price
- Seasonal revenue fluctuations indicate peak tourist seasons

## Dashboard Features
- Interactive filters for exploring different segments
- Multiple visualization types for comprehensive analysis
- Clean, professional layout optimized for insights
